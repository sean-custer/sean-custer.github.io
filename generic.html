Theseus project

An orchestrator for rapidly deployable cloud and edge environments for native Linux applications to a front-end web client hosted in AWS DoD/DHS. Beta target application was the Minotaur Mission System, a sensor-fusing software combining radar, tracking, surveillance, communication and camera data functionality.

Leveraging Terraform and Ansible, our team could rapidly deploy scalable infrastructure as code to the cloud, customizing all configurations from a single node via ansible and bootstrap cloud init scripts.In addition, this project utilized shadowing in Xpra software running in a container to export existing interactive display containing Minotaur to a web client.

Skills utilized

  • Built and deployed secure cloud environments via Amazon Web Services GovCloud to Minotaur clients
  • Produced technical documents explaining how to troubleshoot and reproduce system operations
  • Participated in software product delivery in an agile environment
  • Programmed using Linux OS with JavaScript, C/++, Python and Docker in a legacy codebase
